Introduction

Thank you for visiting. This guide was written to help you get familiar with the BLS Public Data API. If you would like to find official documentation for the BLS API please go to the BLS developer's site.

The BLS Public Data API provides access to employment data. The retrieved information is presented in JSON format.

There exists two versions for the BLS API. The current guide focuses on version 1.0. This version does not require registration, it is free and open to the public. However, it has some limitations comparing to version 2.0. We will not see a comparison of the two versions here. You can easily find this information on the official web site with the link that was provided above. You do not need to obtain any keys to start using version 1.0 of the BLS API.

The official documentation includes the sample code for some popular programming languages. However, it does not cover how to make requests using Node.js and Express. This guide is written to show how to work with the BSL API using mentioned technologies.

The following areas will be covered:

  • using single series signature;
  • using multiple series signature;
  • using single or multiple series signature, specifying years;
  • example of a small application;

We will see the code examples for every section. Hopefully, you find this guide useful.
